## 컴포넌트?

리액트 컴포넌트의 역할
- 마크업을 뿌릴수 있는 자바스크립트 함수
- 재사용성을 높여주는 역할이다.

컴포넌트 빌드 방법
컴포넌트 내보내기 -> 함수정의 **(function ( ) { } 을 사용)** 

컴포넌트 중첩 및 구성
컴포넌트를 정의한 다음 원하는 곳에 여러 번 사용할 수 있다 -> 리액트!
**단 정의를 중첩해서는 안됩니다 (실행시 매우 느리고 버그를 유발한다 -> 랜더링 할때마다 함수안의 함수를 정의하기 때문)**
1. 한 파일 내에서 처리하는 것이 좋은 경우(modal) 
2. 한 컴포넌트 에서만 사용하는 게 확실한 경우
