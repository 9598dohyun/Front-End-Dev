## 👩🏻‍💻 스터디 방식

1. 해당 레포를 자신의 레포로 fork 해주세요 
2. 공부한 내용을 정리해서 upstream으로 해당 레포로 PR 생성 (형식 자유)
3. 각 주차별 폴더에 자기이름(숫자).md의 형식으로 깃허브 업로드
4. 학습 중 궁금한 점 및 소통은 슬랙 채널이나 카톡을 통해 질문및 의견공유
